Atividade desenvolvimeto web

Aluno: Jefferson Nunes Costa

objetivo: Fazer um site que contenha uma pagina de login e cadastro, sobre e um index.

Como foi feito:

1- criar pastas referentes ao projeto.

2- criar um arquivo html denominado login.
Nessa etapa eu criei um form para fazer a caixa de login. Nesse form fiz um input do tipo text para o nome do usuario
e um do tipo password para senha. Um botão para entrar no index e um link para cadastro.

3- criar um arquivo html nomeado cadastro.
do mesmo modo da tela login, criei um form para passar os dados do usuario. Pedindo nome, email e senha. Em seguida fiz o botão cadastrar
que redireciona para a tela login.

4- criar arquivo html index.

criei um menu para o cabeçalho contendo o inicio, sobre e a opção de sair para tela de login. Após isso criei as divs que 
contem as informações do site em sequência.

5- criar arquivo html sobre.

repliquei o menu na tela sobre e escrevi numa tag <h> as informações referentes.

6- vale ressaltar que em todas as telas esta sendo aplicado o design vindo do arquivo estilo.css.
